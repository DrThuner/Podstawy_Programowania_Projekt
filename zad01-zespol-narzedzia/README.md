# Zadanie 1 — Zespoły i narzędzia

## Cel
Podział na zespoły dwuosobowe, założenie konta w darmowym repozytorium git,
założenie konta w systemie edycji dokumentów on-line oraz zapoznanie się
z językiem znaczników Markdown.

## Wykonanie
- **Zespół:** *(wpisz imiona i nazwiska)*
- **Repozytorium git:** *(wklej link)*
- **Dokument on-line (sprawozdanie):** *(wklej link)*
- **Markdown:** opisy zadań w repozytorium napisano w plikach `README.md`
  (składnia Markdown — patrz https://www.markdownguide.org).

To zadanie nie wymaga skryptu — efektem jest samo repozytorium i konta.
